들어가야 할 기능
 - 사진의 정보를 불러와서 지도에 표시해준다
 - 사진 1장을 필수로 등록해야하고 1장 등록시 입력해야할 사항들이 있음
 - 입력해야하는 것은 모두가 필수는 아님
 - 입력해줘야하는 데이터 : 별점 표시, 한줄 코멘트, 음식/장소/등등 택1
 - 입력하지 않은 데이터는 나중에 랜덤하게 질문을 하도록 하여 데이터 완성도를 높여줌
 - 약속 잡기 기능
 - 약속 잡기 기능은 public, private mode가 있음
 - public 모드는 친구로 등록되어 있는 사람들의 앱 상단에 인스타 스토리처럼 생성됨
 - public 모드는 푸시기능이 가능함(앱을 실행하지 않아도 친구들에게 모두 전송됨)
 - private 모드는 친구들 중에서 리스트를 선택하여 보낼 수 있음
 - public / private 모두 약속 잡기 예약을 하면 생성한 사람이 승인을 눌러줘야 함
 - 승인이 된 사람들 또한 public / private 모드로 해당 약속을 공유할 수 있음
 - 약속 잡기 기능은 시간이 지나면 앱에서 삭제됨(히스토리로 넘어감)
 - 장소 공유 기능
 - 친구들 사이에서는 장소를 공유할 수 있음
 - private / public 모드가 있고, private의 경우에는 본인에게만 표시가 되고 남에게 공유되지 않음
 - public 장소 공유는 본인이 남긴 데이터를 모두 남에게 공유 가능(사진의 찍은 날짜는 공유되지 않음, 일부는 공유x)
 - business model 사용자는 어느 누구나 광고를 할 수 있다.
 - 본인이 꼭 운영하지 않아도 된다. 
 - 월 일정 금액을 지불하면 AD표시와 함께 해당 지도에서는 항상 뜨게 된다.
 - 광고는 중첩이 가능하다.(위치기준)
 - 중첩이 된 경우에는 랜덤하게 뜨도록 한다.
 - 한 위치에 여러 정보가 등록되어 있는 경우에는 최신 순으로 표시 3개(해당유저의 프로필정보도 띄워줘야할 듯)
 - 장소 등록 기능
 - 장소 등록 모드상태에서 지도에 위치를 찍거나 주소를 입력하면 해당 위치가 확대되어지면서 정보입력 창이 아래에 뜨도록 한다
 - 장소 등록 모드로 들어가면 지도는 전체화면으로 바뀌게 되고 상단에 주소 입력창이 뜨도록 한다.
 - 핀 선택 또는 주소 입력후 정상 작동시 지도는 줌 레벨이 높아지고, 화면의 40프로만 차지하도록 하고 아래에 데이터 입력창이 생성된다
 - 위치를 정확히 모르는 경우에는 "정확하지 않음 이 근처 또는 이 동네" 라는 체크박스를 준다.
 - 정확히 모르는 경우에는 동단위에서 모음처리하도록 한다.
 - 추후에 친구들이 위치를 변경해 줄 수 있도록 한다.
 - 커플 다이어리 친구목록 중 커플로 등록한 사람과 공유되는 다이어리
 - 커플 다이어리는 양쪽 누구든 등록하게 되면 두 사람에게 모두 보여짐(타인에게 공개 안됨)
 - 그룹 다이어리 : 기획 미정 

디자인 구성
 - 탭 구성 : 하단에 위치 / 홈, 지도, 등록, 내 정보
 - 홈 : 현재 자기 위치 기준 친구들의 공유 목록, 약속 잡기 리스트, 최신 등록 순(발행물), 광고 플레이스
 - 지도 : 홈에 표시된 리스트를 모두 지도에 찍어줌
 - 등록 : 현재 위치 기준 지도보기, 이미지 선택, 카메라 실행 -> 위치 잡고 -> 정보입력
 - 내 정보 : 내 플레이스, 히스토리(시간순으로 정렬), 커플다이어리
